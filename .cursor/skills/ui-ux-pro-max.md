# UI/UX Pro Max スキル

このスキルは、デザインシステムの生成、カラーパレット、フォントの組み合わせ、最新のUIトレンドなど、高度なUI/UXデザイン機能を提供します。

## 機能 (Features)

### 1. デザインシステム (Design Systems)
- **グラスモーフィズム (Glassmorphism)**: ぼかし効果を使った半透明の背景デザイン。
- **ニューモーフィズム (Neumorphism)**: プラスチックのような質感の、柔らかく押し出されたUI要素。
- **クレイモーフィズム (Claymorphism)**: 浮遊感のある影を使った、ふわっとした3D要素。
- **ブルータリズム (Brutalism)**: 太いタイポグラフィを用いた、未加工で大胆な美学。
- **ミニマリズム (Minimalism)**: すっきりとした線、十分な余白、コンテンツへの集中。
- **Bento Grid (弁当箱グリッド)**: お弁当箱にインスパイアされた、グリッドベースのレイアウト。

### 2. カラーパレット (Color Palettes)
- **モダンなグラデーション**: 鮮やかな色同士の滑らかな移行。
- **ダークモード**: 目の疲れを軽減する、コントラストの高いダークテーマ。
- **パステルトーン**: 親しみやすいインターフェース向けの、柔らかく落ち着いた色合い。
- **モノクローム**: 洗練された見た目のための、単色のバリエーション。

### 3. タイポグラフィ (Typography)
- **フォントペアリング**: 検証済みのセリフ体とサンセリフ体の組み合わせ。
- **スケールとリズム**: 調和のとれたレイアウトのための数学的な文字サイズ比率。
- **可読性**: コンテンツに最適な行の長さと高さ。

### 4. コンポーネント (Components)
- **カード**: コンテンツを入れるための多用途なコンテナ。
- **モーダル**: ユーザーの注意を引くダイアログ画面。
- **ナビゲーションバー**: 直感的なサイト内移動の構造。
- **ダッシュボード**: 情報の階層が明確な、データ豊富なインターフェース。

## 使い方 (Usage)

UIデザインを依頼する際は、「グラスモーフィズム風のログインカードを作って」のように希望するスタイルを指定すると、これらの機能を活用できます。
