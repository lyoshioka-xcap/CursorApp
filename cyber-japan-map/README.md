# CYBER JAPAN MAP (日本全国制覇ロードマップ)

日本全国47都道府県の制覇状況を可視化・記録するWebアプリケーションです。
サイバーパンク風のスタイリッシュなデザインで、あなたの旅の記録をローカルに保存します。

## 🌟 主な機能

- **日本地図の可視化**: 訪問済みの都道府県がサイバーパンク風（マゼンタ色）にハイライトされます。
- **制覇率メーター**: 47都道府県中、いくつ制覇したかが一目でわかるプログレスバー。
- **訪問記録の保存**: 都道府県をクリックすると、以下の情報を記録できます。
  - 訪問日
  - 5段階評価（星）
  - メモ・感想
  - 写真のアップロード
- **ローカル保存**: データはすべてブラウザ（IndexedDB）に保存されるため、会員登録不要ですぐに使えます。

## 🛠️ 使用技術 (技術スタック)

- **フレームワーク**: Next.js 15 (App Router)
- **言語**: TypeScript
- **スタイリング**: Tailwind CSS v4
- **データベース**: Dexie.js (IndexedDB)
- **地図ライブラリ**: @react-map/japan
- **UIパーツ**: Lucide React (アイコン), Framer Motion (アニメーション)

## 🚀 ローカルでの起動方法 (Getting Started)

このプロジェクトを自分のパソコンで動かす手順です。

1. 必要なライブラリをインストールします。
   ```bash
   npm install
   ```

2. 開発用サーバーを起動します。
   ```bash
   npm run dev
   ```

3. ブラウザを開き、[http://localhost:3000](http://localhost:3000) にアクセスします。
