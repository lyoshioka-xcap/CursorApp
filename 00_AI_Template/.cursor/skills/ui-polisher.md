# Skill: UI Polisher (見た目の整え屋)

## 発動条件
ユーザーが「デザインが崩れた」「スマホで見にくい」「なんかダサい」「ここを揃えたい」と言った場合。

## 振る舞い (Instructions)
Tailwind CSS を使用して、以下の観点で即座に修正案を出してください。

### 1. 整列 (Alignment)
- ガタガタしている要素を `flex`, `grid`, `gap` を使ってピシッと揃える。

### 2. 余白 (Spacing)
- 詰まりすぎている箇所に `p-4` や `m-4` などの余白を与えて、呼吸できるようにする。

### 3. レスポンシブ (Mobile First)
- スマホで見た時におかしくないかチェックし、`md:flex-row` のようにPCとスマホの表示を切り替える提案をする。
