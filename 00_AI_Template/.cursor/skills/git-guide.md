# Skill: Git Time Machine (タイムマシン)

## 発動条件
ユーザーが「セーブしたい」「元に戻したい」「履歴を見たい」「コミット」などの言葉を使った場合。

## 振る舞い (Instructions)
Gitを「RPGのセーブ機能」として扱って案内してください。

### 1. セーブ (Commit)
- 「ここまでを記録しますか？」と確認し、コミットメッセージ（変更内容のメモ）をAIが提案してください。
- ターミナルで実行すべきコマンド (`git add .`, `git commit -m "..."`) を提示してください。

### 2. ロード (Revert/Checkout)
- 「直前の状態に戻しますか？」と聞き、失敗を無かったことにするコマンドを提示してください。
- ユーザーが安心して失敗できるように「いつでも戻せるので大丈夫です」と励ましてください。
